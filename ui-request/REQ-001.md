# REQ-001: Timeline

## 概要
- **種別**: Component
- **優先度**: Medium
- **依頼元**: Centra

## 既存チェック
- **既存コンポーネント**: 該当なし — 時系列データを表示するコンポーネントが存在しない
- **既存CSSクラス**: 該当なし
- **既存トークン**: `var(--bg-elevated)`, `var(--border-default)`, `var(--text-secondary)` で基本スタイルは対応可能
- **判定**: 新規作成が必要

> **注意**: 上記チェックで既存のもので完全に対応可能な場合、この依頼は作成しないこと。

## 何が必要か
職歴・学歴・イベント履歴などの時系列データを縦型タイムラインとして表示するコンポーネント。現状、Centra の WorkExperienceTimeline で独自実装しているが、汎用的なタイムラインはデザインシステム側で提供すべき。

## 仕様

### Props
```typescript
interface TimelineItem {
  id: string
  title: string
  subtitle?: string
  description?: string
  date: string           // 表示用の日付文字列
  icon?: React.ReactNode // カスタムアイコン
  status?: 'completed' | 'current' | 'upcoming'
}

interface TimelineProps {
  items: TimelineItem[]
  variant?: 'default' | 'compact'
  className?: string
}
```

### ビジュアル仕様
```
  ●── Title 1
  │   Subtitle · Date
  │   Description text here
  │
  ◉── Title 2 (current)
  │   Subtitle · Date
  │   Description text here
  │
  ○── Title 3 (upcoming)
      Subtitle · Date
      Description text here
```

- `●` completed: `var(--accent-primary)` で塗りつぶし
- `◉` current: `var(--accent-primary)` リング + パルスアニメーション
- `○` upcoming: `var(--border-default)` のアウトラインのみ
- 縦線: `var(--border-subtle)` 1px solid

**状態** (5状態ルール):
Timeline 自体は静的表示コンポーネントのため、5状態は省略。
アイテムがクリッカブルな場合は `onClick` prop を追加し、以下を適用:

| 状態 | background | color | border | その他 |
|------|------------|-------|--------|--------|
| Default | transparent | `var(--text-primary)` | — | — |
| Hover | `var(--bg-hover)` | `var(--text-primary)` | — | cursor: pointer |
| Active | `var(--bg-active)` | `var(--text-primary)` | — | scale(0.99) |
| Disabled | transparent | `var(--text-disabled)` | — | opacity: 0.4 |
| Focus | transparent | `var(--text-primary)` | — | outline: 2px solid `var(--accent-primary)` |

### デザイントークン
- ドット背景 (completed): `var(--accent-primary)`
- ドットボーダー (upcoming): `var(--border-default)`
- 縦線: `var(--border-subtle)`
- タイトル: `var(--text-primary)`
- サブタイトル/日付: `var(--text-secondary)`
- 説明文: `var(--text-tertiary)`

### レスポンシブ
- Desktop: そのまま表示
- Mobile: compact variant をデフォルト適用（padding 縮小、description 省略可）

## 使用例
```tsx
import { Timeline } from '@ground/ui'

const items = [
  {
    id: '1',
    title: 'Senior Engineer',
    subtitle: 'Company A',
    date: '2024 - Present',
    status: 'current' as const,
  },
  {
    id: '2',
    title: 'Engineer',
    subtitle: 'Company B',
    date: '2021 - 2024',
    status: 'completed' as const,
  },
]

<Timeline items={items} variant="default" />
```
